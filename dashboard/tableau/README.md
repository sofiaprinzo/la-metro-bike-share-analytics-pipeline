# Tableau Dashboard

This folder contains an earlier Tableau version of the dashboard.

The Tableau dashboard was built from the 2026 Q1 Metro Bike Share data only. The current project dashboard is the Streamlit app, which supports multiple quarters.

Published Tableau dashboard:

[LA Metro Bike Share Demand Dashboard](https://public.tableau.com/shared/2S5YQT764?:display_count=n&:origin=viz_share_link)

## Exported Files

The Tableau files were created from CSV exports in:

```text
exports/tableau/
```

Those exports came from the DuckDB reporting tables for the 2026 Q1 version of the project.
